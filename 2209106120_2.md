   ### 2.8.1 Moving Average Filter

   Moving Average Filter (MAF) adalah filter linear yang menghitung rata-rata
   dari N sampel terakhir untuk menghaluskan sinyal dan mengurangi noise
   frekuensi tinggi.

   Formula:

   y[n] = (1/N) Œ£(i=0 to N-1) x[n-i]

   dimana:
   - y[n] = output filter pada sampel ke-n
   - x[n] = input filter pada sampel ke-n
   - N = ukuran window (jumlah sampel yang dirata-rata)

   Karakteristik:
   - Kompleksitas: O(N) per sampel
   - Latency: (N-1)/2 sampel
   - Atenuasi: -20 dB/decade di stopband
   - Trade-off: N besar ‚Üí sinyal lebih halus tapi latency tinggi

   Pada penelitian ini digunakan N = 5 untuk menyeimbangkan smoothing dan latency.

   ### 2.8.2 Butterworth Low-Pass Filter

   Butterworth Filter adalah filter IIR (Infinite Impulse Response) yang memiliki
   frequency response maksimally flat di passband.

   Formula (2nd order):

   y[n] = b0*x[n] + b1*x[n-1] + b2*x[n-2] - a1*y[n-1] - a2*y[n-2]

   dimana koefisien a dan b dihitung dari cutoff frequency fc dan sampling rate fs.

   Untuk fc = 4 Hz, fs = 50 Hz (sesuai karakteristik detak jantung normal):

   Normalized frequency: wn = 2*pi*fc/fs = 0.5027

   Koefisien filter (dihitung menggunakan scipy.signal.butter):
   b = [b0, b1, b2]
   a = [1, a1, a2]

   Karakteristik:
   - Order: 2 (balance antara atenuasi dan kompleksitas)
   - Cutoff frequency: 4 Hz (max heart rate ~240 BPM)
   - Kompleksitas: O(1) per sampel (fixed computation)
   - Atenuasi: -40 dB/decade di stopband (2x MAF)

   Keunggulan dibanding MAF:
   - Frequency selectivity lebih baik
   - Latency lebih rendah
   - Preserve bentuk gelombang lebih baik

  - H0: Tidak ada perbedaan signifikan antara Moving Average dan Butterworth
  - H1: Ada perbedaan signifikan
- Significance level: Œ± = 0.05

**3. Uji Komparasi Kondisi (ANOVA):**
- Metode: Repeated measures ANOVA
- Faktor: Kondisi (istirahat vs post-exercise)
- Significance level: Œ± = 0.05

**4. Agreement Analysis:**
- Metode: Intraclass Correlation Coefficient (ICC)
- Target: ICC > 0.90 (excellent agreement)

#### 3.6.3 Kriteria Penerimaan Sistem

Sistem dianggap **BERHASIL** jika memenuhi:

1. **Akurasi:**
   - MAE < 5 BPM pada kondisi istirahat
   - MAE < 7 BPM pada kondisi post-exercise
   - Pearson r > 0.90 dengan pulse oximeter

2. **Konsistensi:**
   - ICC > 0.85 across responden
   - CV (Coefficient of Variation) < 10%

3. **Efisiensi:**
   - Execution time < 20 ms per sampel
   - End-to-end latency < 200 ms
   - Dapat berjalan continuous selama >1 jam tanpa crash

4. **Komparasi Algoritma:**
   - Setidaknya satu algoritma memenuhi kriteria #1 dan #2
   - Jika keduanya memenuhi, pilih yang lebih efisien (execution time lebih rendah)

#### 3.6.4 Rencana Analisis Data

**Software yang Digunakan:**

- SPSS / R (untuk uji statistik lanjutan)
- Microsoft Excel (untuk tabulasi awal)

**Tahapan Analisis:**

1. **Data Cleaning:**
   - Remove outliers (BPM < 40 atau > 200)

   - Validate data integrity

2. **Deskriptif Statistik:**
   - Mean, median, SD, min, max untuk setiap kondisi dan algoritma
   - Visualisasi: boxplot, histogram, time series plot

3. **Inferential Statistics:**
