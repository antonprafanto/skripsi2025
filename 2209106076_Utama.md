2.1.2 Justifikasi Pemilihan Algoritma Decision Tree CART

   rendah, cocok untuk aplikasi real-time pada mikrokontroler IoT.

#### 3.6 Perancangan Pengujian

**Harus dijelaskan dengan detail:**

**1. Pengujian Akurasi Model:**

- **Metode**: K-Fold Cross Validation (k=5 atau k=10)
- **Pembagian Data**: 80% training, 20% testing
- **Metrics Evaluasi**:

  a) **Confusion Matrix:**
  ```
                Predicted
                Baik  Perlu  Buruk
  Actual Baik    TP    FP     FP
         Perlu   FN    TP     FP
         Buruk   FN    FN     TP
  ```

  b) **Akurasi:**
  ```
  Akurasi = (TP + TN) / (TP + TN + FP + FN) √ó 100%  ............(3.1)
  ```

  c) **Precision:**
  ```
  Precision = TP / (TP + FP) √ó 100%  ..........................(3.2)
  ```

  d) **Recall:**
  ```
  Recall = TP / (TP + FN) √ó 100%  .............................(3.3)
  ```

  e) **F1-Score:**
  ```
  F1-Score = 2 √ó (Precision √ó Recall) / (Precision + Recall)  ..(3.4)
  ```
