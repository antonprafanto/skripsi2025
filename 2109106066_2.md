**10. Formula Defuzzifikasi Tidak Eksplisit**
- **Lokasi:** Bagian Defuzzifikasi
- **Masalah:** Rumus weighted average tidak dituliskan dengan jelas
- **Solusi:** Tambahkan formula:

$$Z = \frac{\sum_{i=1}^{n} \alpha_i \times z_i}{\sum_{i=1}^{n} \alpha_i}$$

Dimana:
- Z = Output defuzzifikasi
- Î±i = Nilai Î±-predikat rule ke-i
- zi = Nilai konsekuen rule ke-i (1, 2, atau 3)
- n = Jumlah rules yang aktif

### BAB IV - Hasil dan Pembahasan (Halaman 61-80)

**13. Penggunaan MAE untuk Output Kategorikal**
- **Lokasi:** Subbab Pengujian Akurasi
- **Masalah:** MAE (Mean Absolute Error) digunakan untuk menguji akurasi output kategorikal (Rendah/Sedang/Tinggi). MAE seharusnya untuk data kontinu.
- **Solusi:** Gunakan **Confusion Matrix** untuk validasi klasifikasi:

| | Prediksi Rendah | Prediksi Sedang | Prediksi Tinggi |
|---|---|---|---|
| **Aktual Rendah** | True Negative | False Positive | False Positive |
| **Aktual Sedang** | False Negative | True Positive | False Positive |
| **Aktual Tinggi** | False Negative | False Negative | True Positive |

Kemudian hitung:
- **Akurasi** = (TP + TN) / Total Ã— 100%
- **Precision** = TP / (TP + FP)
- **Recall** = TP / (TP + FN)

**14. Jumlah Data Tidak Representatif**
- **Lokasi:** Tabel 4.1 Data Pengujian
- **Masalah:** Hanya 50 sampel data, semua menunjukkan kondisi normal/stabil
- **Solusi:**
  - Tambah jumlah data minimal 100 sampel

  - Jika tidak bisa mendapat data real, gunakan simulasi dengan variasi nilai
