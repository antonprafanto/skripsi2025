## üéì KESIMPULAN

### Mengapa Saat Ini Lebih Cocok untuk Sistem Informasi?

1. **Fokus pada Manajemen Data**
   - CRUD barang, transaksi masuk/keluar
   - Laporan stok
   - Tidak ada pemrosesan otomatis/cerdas

2. **Tidak Ada Komponen Algoritma**
   - Hanya monitoring pasif
   - Tidak ada prediksi/forecasting
   - Tidak ada optimasi
   - Tidak ada machine learning

3. **Pengujian Hanya Fungsional**
   - Black box testing ‚Üí menguji fitur aplikasi
   - Usability testing ‚Üí menguji kemudahan penggunaan
   - Tidak ada evaluasi akurasi algoritma
   - Tidak ada analisis performa komputasi

4. **Metodologi RUP Saja Tidak Cukup**
   - RUP hanya metodologi software engineering (umum)
   - Tidak ada tahapan pengembangan algoritma
   - Tidak ada tahapan training/testing model
