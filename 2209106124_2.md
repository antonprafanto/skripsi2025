### C. Koreksi Keterangan Confusion Matrix (hal 14)

**Salah (di proposal hal 14):**
```
TP : True Positive     FN : False Positives
FN : False Negatives   TN : True Negatives
```

**Benar:**
```
TP : True Positive     FP : False Positive
FN : False Negative    TN : True Negative
```

**Penjelasan:**
- FN (False Negatives) muncul DUA KALI di baris kiri
- FP (False Positive) TIDAK ADA sama sekali
- Kolom kanan seharusnya FP dan TN, bukan FN dan TN
- Ini kesalahan konseptual yang menunjukkan kurang teliti dalam penulisan
