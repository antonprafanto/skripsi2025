##### Pseudocode Peak Detection (hal. 61)

**MASALAH:** Konstanta tidak didefinisikan

```
PROCEDURE detectPeak(value, timestamp):
  IF value > adaptive_threshold AND
     (timestamp - lastPeakTime) > MIN_PEAK_DISTANCE_MS THEN
    record peak_time
    RETURN TRUE
  END IF
END PROCEDURE
```

**PERBAIKAN:**

```
CONSTANTS:
  MIN_PEAK_DISTANCE_MS = 400        // Minimal 400ms antar peak (max 150 BPM)
  THRESHOLD_PERCENTILE = 0.7        // 70% dari max signal
  WINDOW_SIZE = 100                 // 100 sampel untuk adaptive threshold

PROCEDURE updateThreshold(signal_buffer[]):
  max_value = MAX(signal_buffer)
  min_value = MIN(signal_buffer)
  adaptive_threshold = min_value + THRESHOLD_PERCENTILE * (max_value - min_value)
  RETURN adaptive_threshold
END PROCEDURE

PROCEDURE detectPeak(value, timestamp, adaptive_threshold, lastPeakTime):
  IF value > adaptive_threshold AND
     (timestamp - lastPeakTime) > MIN_PEAK_DISTANCE_MS THEN
    lastPeakTime = timestamp
    RETURN TRUE
  ELSE
    RETURN FALSE
  END IF
END PROCEDURE

PROCEDURE calculateBPM(peak_times[]):
  IF LENGTH(peak_times) < 2 THEN
    RETURN 0
  END IF

  intervals[] = []
  FOR i = 1 TO LENGTH(peak_times)-1 DO
    interval = peak_times[i] - peak_times[i-1]
    intervals.APPEND(interval)
  END FOR

  avg_interval = MEAN(intervals)  // dalam milidetik
  bpm = 60000 / avg_interval

  RETURN bpm
END PROCEDURE
```
