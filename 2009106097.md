```
ABSTRAK

Usaha Mikro, Kecil, dan Menengah (UMKM) memiliki peran penting dalam ekonomi
Indonesia dengan kontribusi lebih dari 60% terhadap PDB nasional. Namun, banyak
UMKM masih mengelola pemesanan secara manual yang membutuhkan waktu 5-10 menit
per pesanan dan rentan kesalahan pencatatan. Penelitian ini bertujuan mengembangkan
sistem integrasi chatbot WhatsApp dengan printer thermal berbasis Internet of Things
untuk meningkatkan efisiensi pemesanan di UMKM Bolu Tape Mbah Kong. Metode yang
digunakan adalah System Development Life Cycle (SDLC) model Waterfall dengan tahapan
analisis kebutuhan, perancangan, implementasi, pengujian, dan pemeliharaan. Sistem
menggunakan RASA sebagai framework Natural Language Processing untuk chatbot, Laravel
sebagai backend dan database, protokol MQTT untuk komunikasi IoT, serta ESP32 sebagai
mikrokontroler yang terhubung dengan printer thermal melalui platform Thinger.io.
Pengujian dilakukan menggunakan Black Box Testing untuk validasi fungsional, System
Usability Scale (SUS) untuk pengukuran kebergunaan, ISO/IEC 25010 untuk kualitas
sistem, dan Return On Investment (ROI) untuk analisis ekonomis. Hasil yang diharapkan
adalah pengurangan waktu pemrosesan pesanan menjadi kurang dari 5 menit, peningkatan
akurasi pencatatan pesanan, dan efisiensi operasional bagi pemilik UMKM.

Kata kunci: Chatbot, Internet of Things, UMKM, Natural Language Processing, RASA,
ESP32, WhatsApp, Sistem Pemesanan, Waterfall

ABSTRACT

Micro, Small, and Medium Enterprises (MSMEs) play a vital role in Indonesia's economy,
contributing over 60% to the national GDP. However, many MSMEs still manage orders
manually, requiring 5-10 minutes per order and being prone to recording errors. This
research aims to develop an integrated system of WhatsApp chatbot with thermal
printer based on Internet of Things to improve ordering efficiency at UMKM Bolu Tape
Mbah Kong. The method used is System Development Life Cycle (SDLC) Waterfall model
with stages of requirements analysis, design, implementation, testing, and maintenance.
The system uses RASA as Natural Language Processing framework for chatbot, Laravel
as backend and database, MQTT protocol for IoT communication, and ESP32 as
microcontroller connected to thermal printer through Thinger.io platform. Testing
is conducted using Black Box Testing for functional validation, System Usability
Scale (SUS) for usability measurement, ISO/IEC 25010 for system quality, and Return
On Investment (ROI) for economic analysis. Expected results include reduction of
order processing time to less than 5 minutes, improved order recording accuracy,
and operational efficiency for MSME owners.

Keywords: Chatbot, Internet of Things, MSME, Natural Language Processing, RASA,
ESP32, WhatsApp, Ordering System, Waterfall
```
