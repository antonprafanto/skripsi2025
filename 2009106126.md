   - Memiliki smartphone Android
   - Usia 15-40 tahun
   - Berdomisili di Samarinda
   - Bersedia menjadi responden

   Ukuran Sampel:
   Menggunakan rumus Slovin dengan margin error 15%:

   n = N / (1 + N * e²)
   n = 1000 / (1 + 1000 * 0.15²)

   n = jumlah sampel
